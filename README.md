# GitHub Profile Generator

A Command-line application that dynamically generates a PDF profile from a GitHub username.

## Description
The user will be prompted for a favorite color, which will be used as the background color for cards.

The PDF will be populated with the following:

* Profile image
* User name
* Links to the following:
  * User location via Google Maps 
  * User GitHub profile 
  * User blog 
* User bio
* Number of public repositories
* Number of followers
* Number of GitHub stars
* Number of users following

## Installation


```bash
- Clone repo to your machine 'git clone https://github.com/mesayb/GitHub_PDF_Generator.git'
- Install dependecies using 'npm install' or 'npm i'
- To start generating profile use command 'node index.js' 
you can use the demo attached for more detailed how to use


https://github.com/mesayb/GitHub_PDF_Generator/blob/master/GitHubUserProfileInfoPDF.gif?raw=true
```
